# AI Automation — учебный курс

Учебный репозиторий по дорожной карте «AI Automation: от нуля до фриланса на Upwork».

Содержит код упражнений по Фазам 0–1 (Python-фундамент и LLM API).
Каждая папка `module-XX-...` — отдельный модуль курса.

Проектные репозитории (Invoice Processor, Price Monitor и т.д.) живут отдельно.

## Установка

```bash
git clone <url-репозитория>
cd ai-automation-course
python3 -m venv venv
source venv/bin/activate
